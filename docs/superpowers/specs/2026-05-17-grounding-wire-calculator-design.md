# 接地線太さ計算アプリ 設計書

**日付**: 2026-05-17  
**対象プロジェクト**: 接地線太さ計算ツール  
**形態**: Next.js + Vercel デプロイ

---

## 概要

現場の電気工事士がスマホで接地線の最小太さを即座に計算できるWebアプリ。  
A種・B種・C種・D種接地に対応し、電技解釈と内線規程を切り替えて使える。

---

## 要件まとめ

| 項目 | 内容 |
|---|---|
| 対応接地種別 | A種・B種・C種・D種（全4種） |
| 適用規格 | 電技解釈 / 内線規程（切り替え式） |
| 主な使用環境 | 現場スマホ（スマホファースト） |
| 出力内容 | 最小線径(mm²) + 接地抵抗値 + 根拠条文 |
| デプロイ先 | Vercel（Next.js） |

## 計算ロジック

### 全種共通の基本式（内線規程 資料1-3-6）

```
断面積[mm²] = 0.052 × 該当電流[A]
```

計算結果は規格品の直近上位サイズに切り上げる。  
規格品サイズ: 2.0 / 3.5 / 5.5 / 8 / 14 / 22 / 38 / 60 / 100 mm²

---

### A種接地工事（電技解釈第17条第1項 / 内線規程1350-4表）

- **入力**: 過電流遮断器の定格電流 (A)
- **計算式**: 断面積 = 0.052 × 定格電流
- **最低値**: 5.5mm²（直径2.6mm以上）
- **接地抵抗**: 10Ω以下

### B種接地工事（電技解釈第17条第2項・第24条 / 内線規程1350-5表）

- **入力**: 変圧器容量 (kVA) ＋ 二次側電圧 (V) ＋ 相数（単相／三相）
- **二次定格電流の計算**:
  - 三相: `定格電流 = 容量(kVA)×1000 ÷ (√3 × 電圧V)`
  - 単相: `定格電流 = 容量(kVA)×1000 ÷ 電圧V`
- **接地線太さ**: 断面積 = 0.052 × 二次定格電流
- **最低値**: 5.5mm²（直径2.6mm以上）
- **接地抵抗**: 150 / IG (Ω) 以下（IG = 高圧側1線地絡電流）

### C種接地工事（電技解釈第17条第3項 / 内線規程1350-3表）

- **入力**: 機器・電路の定格電流 (A)
- **計算式**: 断面積 = 0.052 × 定格電流
- **最低値**: 2.0mm²（直径1.6mm以上）
- **接地抵抗**: 10Ω以下（漏電遮断器付きは500Ω以下）

### D種接地工事（電技解釈第17条第4項 / 内線規程1350-3表）

- **入力**: 機器・電路の定格電流 (A)
- **計算式**: C種と同式（断面積 = 0.052 × 定格電流）
- **最低値**: 2.0mm²（直径1.6mm以上）
- **接地抵抗**: 100Ω以下（漏電遮断器付きは500Ω以下）

## コンポーネント詳細

### `lib/calculator.ts`

```typescript
type GroundType = 'A' | 'B' | 'C' | 'D'
type Phase = 'single' | 'three'

interface CalculatorInput {
  type: GroundType
  // A種: 過電流遮断器の定格電流
  // B種: 変圧器容量(kVA) + 二次側電圧(V) + 相数
  // C/D種: 機器・電路の定格電流
  breakerCurrent?: number    // A種用 (A)
  transformerKva?: number    // B種用 (kVA)
  secondaryVoltage?: number  // B種用 (V)
  phase?: Phase              // B種用
  ratedCurrent?: number      // C/D種用 (A)
}

interface CalculatorResult {
  minCrossSection: number  // mm² (規格品切り上げ後)
  rawCrossSection: number  // mm² (計算値そのまま)
  maxResistance: string    // "10Ω以下" など
  legalBasis: string       // 根拠条文テキスト
  note?: string            // 補足（B種の接地抵抗計算値など）
}

export function calculate(input: CalculatorInput): CalculatorResult
export function roundUpToStandard(raw: number): number // 規格品切り上げ
```

### `components/InputForm.tsx`

種別ごとに表示項目を動的切り替え：
- A種: 過電流遮断器の定格電流 (A)
- B種: 変圧器容量 (kVA) ＋ 二次側電圧 (V) ＋ 単相／三相
- C種: 機器・電路の定格電流 (A)
- D種: 機器・電路の定格電流 (A)（C種と同形式）

### `components/ResultCard.tsx`

計算結果を見やすく表示。未計算時は非表示。  
結果エリアは緑背景で視覚的に分かりやすく。

## 非機能要件

- **スマホファースト**: max-width 390px でデザイン、PC でも崩れない
- **オフライン動作**: 計算ロジックはクライアントサイドのみ（API不要）
- **レスポンス速度**: 入力完了後、計算ボタン押下で即時表示
- **型安全**: TypeScript strict モード
