# 接地線太さ計算アプリ 設計書

**日付**: 2026-05-17  
**対象プロジェクト**: 接地線太さ計算ツール  
**形態**: Next.js + Vercel デプロイ

---

## 概要

現場の電気工事士がスマホで接地線の最小太さを即座に計算できるWebアプリ。  
A種・B種・C種・D種接地に対応し、電技解釈と内線規程を切り替えて使える。

---

## 要件まとめ

| 項目 | 内容 |
|---|---|
| 対応接地種別 | A種・B種・C種・D種（全4種） |
| 適用規格 | 電技解釈 / 内線規程（切り替え式） |
| 主な使用環境 | 現場スマホ（スマホファースト） |
| 出力内容 | 最小線径(mm²) + 接地抵抗値 + 根拠条文 |
| デプロイ先 | Vercel（Next.js） |

## 計算ロジック

### A種接地工事（電技解釈第17条第1項）

- 接地線最小太さ: **5.5mm²以上**（直径2.6mm以上）
- 高圧・特別高圧電路に施す場合は地絡電流 IG から判定:

| IG (A) | 最小断面積 |
|--------|-----------|
| 〜5 | 5.5mm² |
| 5〜10 | 14mm² |
| 10〜20 | 22mm² |
| 20〜40 | 38mm² |
| 40〜 | 60mm² |

### B種接地工事（電技解釈第24条 / 内線規程1340-1節）

**電技解釈**: 接地抵抗値 = 150 / IG (Ω) 以下  
接地線太さは変圧器容量（kVA）から：

| 変圧器容量 | 最小断面積 |
|-----------|-----------|
| 〜75kVA | 14mm² |
| 〜150kVA | 22mm² |
| 〜300kVA | 38mm² |
| 300kVA超 | 60mm² |

**内線規程**: 上記表に準じるが、遮断時間により緩和規定あり（1秒以内: × 1/2）

### C種接地工事（電技解釈第17条第3項）

- 最小太さ: **5.5mm²以上**（直径2.6mm以上）
- 接地抵抗: 10Ω以下（漏電遮断器付きは500Ω以下）
- 入力: 定格電流（表示参考用）

### D種接地工事（電技解釈第17条第4項）

- 最小太さ: **2.0mm²以上**（直径1.6mm以上）
- 接地抵抗: 100Ω以下（漏電遮断器付きは500Ω以下）
- 入力不要（固定値）

## コンポーネント詳細

### `lib/calculator.ts`

```typescript
type GroundType = 'A' | 'B' | 'C' | 'D'
type Standard = 'denki' | 'naisen'

interface CalculatorInput {
  type: GroundType
  standard: Standard
  faultCurrent?: number   // IG (A) — A種・B種
  transformerKva?: number // 変圧器容量 — B種
  ratedCurrent?: number   // 定格電流 — C種
}

interface CalculatorResult {
  minCrossSection: number  // mm²
  maxResistance: string    // "10Ω以下" など
  legalBasis: string       // 条文テキスト
  note?: string            // 補足
}

export function calculate(input: CalculatorInput): CalculatorResult
```

### `components/InputForm.tsx`

種別ごとに表示項目を動的切り替え：
- A種: IG入力フィールド
- B種: 変圧器容量 OR IG（どちらか選択）
- C種: 定格電流（参考表示用）
- D種: フィールドなし（固定値を即表示）

### `components/ResultCard.tsx`

計算結果を見やすく表示。未計算時は非表示。  
結果エリアは緑背景で視覚的に分かりやすく。

## 非機能要件

- **スマホファースト**: max-width 390px でデザイン、PC でも崩れない
- **オフライン動作**: 計算ロジックはクライアントサイドのみ（API不要）
- **レスポンス速度**: 入力完了後、計算ボタン押下で即時表示
- **型安全**: TypeScript strict モード
